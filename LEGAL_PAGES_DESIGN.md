## Design Specifications

### Colors

- **Header Background:** `#fff` (white)
- **Header Text:** `#333` (dark gray)
- **Section Titles:** `#333`, weight `600` (semi-bold)
- **Body Text:** `#444`, line-height `24`
- **Last Updated:** `#666`, italic
- **Email Links:** `#4ECDC4` (brand teal), underlined
- **Footer Text:** `#666`, centered
- **Border Colors:** `#eee` (light gray)

### Typography

- **Header Title:** 18px, weight 600
- **Section Titles:** 18px, weight 600
- **Body Text:** 15px, regular
- **Last Updated:** 13px, italic
- **Footer:** 13px, regular

### Spacing

- **Screen Padding:** 20px
- **Section Title Margin Top:** 24px
- **Section Title Margin Bottom:** 12px
- **Paragraph Margin Bottom:** 16px
- **Bullet Padding Left:** 8px
- **Footer Top Margin:** 32px
- **Footer Top Padding:** 24px

### Interactive Elements

#### Email Links

```jsx
<TouchableOpacity onPress={() => Linking.openURL("mailto:privacy@wihngo.com")}>
  <Text style={styles.link}>privacy@wihngo.com</Text>
</TouchableOpacity>
```

#### Back Button

```jsx
<TouchableOpacity onPress={() => router.back()} style={styles.backButton}>
  <FontAwesome6 name="chevron-left" size={20} color="#333" />
</TouchableOpacity>
```

## Key Differences from Web Version

| Aspect      | Web Version       | Mobile App Version              |
| ----------- | ----------------- | ------------------------------- |
| Navigation  | Direct page URLs  | App router with screens         |
| Header      | Simple text       | Custom header with back button  |
| Layout      | Browser scroll    | React Native ScrollView         |
| Links       | HTML `<a>` tags   | TouchableOpacity + Linking API  |
| Typography  | CSS styles        | StyleSheet with specific values |
| Email Links | `mailto:` href    | Linking.openURL()               |
| Responsive  | CSS media queries | Native mobile layout            |
| Languages   | Single language   | 16 languages with i18n          |
