# Wihngo Mobile App
**Wihngo - Production-Grade Mobile Platform Demo**
[https://www.loom.com/share/bfd1b8db0a8949d9b2ff541363cb997f](https://www.loom.com/share/bfd1b8db0a8949d9b2ff541363cb997f)
Production-grade cross-platform mobile application built with Expo React Native as part of the Wihngo platform case study.

## Overview
Wihngo is a mobile-first platform designed to support content-driven communities.  
This repository represents the mobile client, built with scalability, performance, and real-world production constraints in mind.

## Tech Stack
- Expo React Native
- TypeScript
- Zustand / Context (state management)
- REST API integration
- Secure media handling
- Modular, scalable project structure

## Key Features
- Cross-platform support (Android / iOS)
- Media-rich content feeds
- Secure authentication flow
- Audio recording and upload
- AI-ready integrations via backend
- Error handling and network resilience

## Architecture Notes
- Clean separation between UI, domain logic, and API services
- Reusable components and screens
- Environment-based configuration
- Optimized for maintainability and future scaling

## Why Expo?
Expo was chosen to accelerate development, simplify native integrations, and ensure consistent behavior across platforms while maintaining performance.
